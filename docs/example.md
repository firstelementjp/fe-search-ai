# Examples

## Documentation site

Use FE Search AI to help visitors find pages in a documentation site. Index documentation pages, add the chat UI to the documentation top page, and test common support questions.

## Product support

For product websites, index product pages, FAQ articles, and support documents. Visitors can ask questions in natural language instead of guessing keywords.

## Knowledge base

For internal or public knowledge bases, combine custom post types and taxonomies with RAG search so users can ask direct questions about policies, procedures, or content.

## Japanese website

Enable Japanese tokenization and test both exact keyword questions and natural language questions. If exact term matching is important, make sure key terms appear in titles and headings.

## Setup checklist

- Configure providers.
- Connect Qdrant.
- Select sync targets.
- Run sync.
- Add chat UI.
- Test and tune prompts.
