# Contributing

Thank you for your interest in FE Search AI.

## Before contributing

- Open an issue for major changes.
- Keep changes focused and easy to review.
- Follow WordPress Coding Standards where applicable.
- Add or update documentation when behavior changes.

## Development setup

Install PHP and Node dependencies before running checks.

```bash
composer install
npm install
```

## Useful commands

```bash
composer phpcs
npm run lint
npm run format:check
npm run build
```

## Pull requests

A good pull request includes:

- Clear description of the change
- Testing notes
- Screenshots for UI changes
- Documentation updates if needed

## Security issues

Do not open public issues for vulnerabilities. Follow the repository security policy.

[Security Policy](https://github.com/firstelementjp/fe-search-ai/security/policy)
