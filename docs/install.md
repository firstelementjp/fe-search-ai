# Installation

## Requirements

- WordPress 6.6 or higher
- PHP 7.4 or higher
- MySQL 5.7 or higher
- OpenSSL extension
- API key for at least one supported AI provider
- Qdrant Cloud or self-hosted Qdrant

## Install from WordPress.org

1. Open **Plugins → Add New** in WordPress admin.
2. Search for **FE Search AI**.
3. Click **Install Now**.
4. Activate the plugin.
5. Open **FE Search AI** from the admin menu.

## Manual installation

1. Download the latest release ZIP from GitHub.
2. Open **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP file.
4. Activate the plugin.

[Download latest release](https://github.com/firstelementjp/fe-search-ai/releases)

## After activation

After activation, open the settings screen and configure these items first:

- Chat provider API key
- Embedding provider API key
- Qdrant endpoint and API key
- Sync target post types
- Chat UI display mode

## Pro add-on

FE Search AI Pro is installed as a separate add-on plugin. Install and activate the free plugin first, then install FE Search AI Pro if you need Pro features.
