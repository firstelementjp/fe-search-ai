# 検索UIの設置

FE Search AIは、WordPressページに埋め込めるチャット型検索UIを提供します。

## ショートコード

投稿、固定ページ、ブロックエディタ内に以下のショートコードを追加します。

```text
[fe-search-ai]
```

## フローティングチャット

表示設定からフローティングチャットを有効化できます。サイト全体でAI検索を利用できるようにしたい場合に便利です。

## 設置に向いている場所

- ドキュメントのトップページ
- ヘルプセンターやFAQページ
- 製品サポートページ
- ナレッジベース
- 訪問者から質問が多いランディングページ

## 回答の流れ

チャットUIは訪問者の質問をWordPressに送信し、インデックス済みコンテンツから関連情報を取得したうえで、AIによる回答を返します。

## 回答品質を調整する

回答が広すぎる、または不正確な場合は、以下を確認してください。

1. 関連コンテンツが同期されているか確認します。
2. 同期対象の設定を見直します。
3. システムプロンプトを調整します。
4. 必要に応じてRerankを有効化します。
5. 元コンテンツのタイトルや本文を改善します。

## テーマ互換性

標準的なWordPressテーマで動作するよう設計されています。テーマ側のCSSリセットが強い場合は、チャット色やレイアウト設定、カスタムCSSで調整してください。
