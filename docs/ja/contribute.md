# 貢献

FE Search AIへの貢献に関心をお寄せいただきありがとうございます。

## 貢献前に

- 大きな変更は事前にIssueで相談してください。
- 変更範囲はレビューしやすい単位にしてください。
- WordPress Coding Standardsに従ってください。
- 挙動が変わる場合はドキュメントも更新してください。

## 開発環境

PHPとNode.jsの依存関係をインストールします。

```bash
composer install
npm install
```

## よく使うコマンド

```bash
composer phpcs
npm run lint
npm run format:check
npm run build
```

## Pull Request

良いPull Requestには以下が含まれます。

- 変更内容の明確な説明
- テスト内容
- UI変更の場合はスクリーンショット
- 必要に応じたドキュメント更新

## セキュリティ問題

脆弱性は公開Issueに投稿しないでください。リポジトリのセキュリティポリシーに従って報告してください。

[Security Policy](https://github.com/firstelementjp/fe-search-ai/security/policy)
