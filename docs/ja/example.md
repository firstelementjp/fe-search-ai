# 例

## ドキュメントサイト

ドキュメントページを探しやすくするためにFE Search AIを利用できます。ドキュメントページをインデックスし、トップページにチャットUIを設置して、よくある質問をテストします。

## 製品サポート

製品サイトでは、製品ページ、FAQ、サポート記事をインデックスします。訪問者はキーワードを推測せず、自然な文章で質問できます。

## ナレッジベース

社内向けまたは公開ナレッジベースでは、カスタム投稿タイプやタクソノミーをRAG検索と組み合わせ、手順やポリシーに関する質問へ回答できます。

## 日本語サイト

日本語トークナイズを有効化し、完全一致のキーワード質問と自然文の質問の両方をテストします。重要語はタイトルや見出しにも含めると効果的です。

## ショートコード例

```text
[fe-search-ai]
```

## セットアップチェックリスト

- プロバイダーを設定します。
- Qdrantを接続します。
- 同期対象を選択します。
- 同期を実行します。
- チャットUIを設置します。
- テストしてプロンプトを調整します。
