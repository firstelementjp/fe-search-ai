# トラブルシューティング

## チャットが回答しない

まず以下を確認してください。

1. チャットプロバイダーのAPIキーが設定されている。
2. EmbeddingプロバイダーのAPIキーが設定されている。
3. Qdrant設定が正しい。
4. コンテンツ同期が完了している。
5. フロントエンドに `[fe-search-ai]` がある、またはフローティングチャットが有効になっている。

## 回答の関連性が低い

- 同期設定を変更した後はインデックスを再構築します。
- 関連する投稿が同期対象に含まれているか確認します。
- 元コンテンツのタイトルや見出しを改善します。
- 必要に応じてCohere Rerankを有効化します。
- システムプロンプトを調整します。

## 同期に失敗する

- バッチサイズを小さくします。
- サーバーのメモリ上限とタイムアウト設定を確認します。
- プロバイダーのAPI利用枠を確認します。
- QdrantのコレクションとEmbeddingモデルのベクトルサイズが一致しているか確認します。

## APIキーテストに失敗する

- キーに余分な空白がないか確認します。
- プロバイダーアカウントの請求設定と利用枠を確認します。
- 選択したモデルがアカウントで利用可能か確認します。
- プロバイダー側のログがあれば確認します。

## 日本語検索の品質が低い

- サイトのロケールとトークナイザー設定を確認します。
- TinySegmenterで不足する場合はYahoo! JAPAN 日本語形態素解析APIを試します。
- 重要な用語はタイトルや見出しに明示します。
